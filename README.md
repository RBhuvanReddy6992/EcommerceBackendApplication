# E-commerce Backend (Microservices Architecture)

This project is an E-commerce Backend application developed using Java and Spring Boot, following a Microservices Architecture. It is designed to handle core functionalities such as user management, product management, and order processing in a scalable and modular way.

Project Status: Under Development
This project is currently in progress, and new features are being added continuously.

Features:

* User Service (Registration and Authentication)
* Product Service (Add, Update, View Products)
* Order Service (Place and Manage Orders)
* API Gateway for routing requests
* Service-to-Service communication using REST/Feign Client
* MySQL database integration
* Kafka-based event-driven communication (in progress)

Architecture:
The project follows a Microservices Architecture where each service is developed and deployed independently.

Tech Stack:

* Java
* Spring Boot
* Spring Security
* MySQL
* Apache Kafka
* REST APIs
* Maven

Project Structure:
EcommerceBackend/

* user-service/
* product-service/
* order-service/
* api-gateway/

Objective:
The goal of this project is to build a scalable backend system and gain practical experience in microservices, distributed systems, and backend development.

Future Enhancements:

* Payment integration
* Notification service
* Docker and Kubernetes deployment
* CI/CD pipeline
* Frontend integration
